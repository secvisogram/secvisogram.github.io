# Acknowledgments - Spezifikation

List of Acknowledgments (`acknowledgments_t`) type instance of value type `array` with 1 or more elements contain a list of `Acknowledgment` elements.

```javascript
"acknowledgments_t": {
  // ...
  "items": {
    // ...
  }
}
```
