# Notes - Spezifikation

Liste von Notizen (`notes_t`) vom Werttyp Array mit 1 oder mehr Elementen vom Typ `Note` enthält Notizen, die für den aktuellen Kontext spezifisch sind.

```javascript
"notes_t": {
  // ...
  "items": {
    // ...
  }
}
```
