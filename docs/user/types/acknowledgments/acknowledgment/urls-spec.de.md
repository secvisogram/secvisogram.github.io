# URLs - Spezifikation

List of URLs (`urls`) of acknowledgment ist ein Container (Werttyp `array`) für 1 oder mehrere `string` vom Typ URL, der eine Liste von URLs oder den Ort der zu bestätigenden Referenz angibt.
