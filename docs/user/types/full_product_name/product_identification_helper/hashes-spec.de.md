# Hashes - Spezifikation

Liste von Hashes (`hashes`) des Wertetyps `array` mit mindestens einem Element enthält eine Liste von kryptografischen Hashes, die zur Identifizierung von Dateien verwendet werden können.

```javascript
"hashes": {
  // ...
  "items": {
    // ...
  }
}
```
