# File Hashes - Spezifikation

Liste von Datei-Hashes (`file_hashes`) vom Wertetyp `array` mit mindestens einem Element enthält eine Liste von kryptografischen Hashes für diese Datei.

```javascript
"file_hashes": {
  // ...
  "items": {
    // ...
  }
}
```
